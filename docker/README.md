# Authensor Control Plane

Enterprise-grade AI agent safety, completely free.

## Quick Start

```bash
docker compose up -d
```

Or pull directly:

```bash
docker pull authensor/control-plane
```

## What's Included
- Policy engine with declarative YAML rules
- Approval workflows (Slack, email, SMS)
- SHA-256 hash-chained audit receipts
- Aegis content safety scanning
- Sentinel real-time monitoring
- PostgreSQL for persistence
